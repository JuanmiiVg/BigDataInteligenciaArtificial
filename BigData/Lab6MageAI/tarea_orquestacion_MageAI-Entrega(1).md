# 📄 Documento de Entrega — Práctica Mage AI

## 🧠 Práctica: Orquestación de pipelines con Mage AI

### 🎯 Objetivo

Desarrollar un pipeline de datos utilizando **Mage AI** que permita:

* Cargar un dataset real
* Aplicar transformaciones
* Generar un resultado final exportado

---

## 📦 Entregables

El alumno deberá entregar:

### 1. Código del pipeline

* Proyecto Mage completo (carpeta o repo)
* Bloques implementados:

  * `Data Loader`
  * `Transformer`
  * `Data Exporter`

---

### 2. Dataset utilizado

* Fichero `.csv`
* O referencia clara si se descarga automáticamente

---

### 3. Evidencias (OBLIGATORIO)

Capturas de pantalla de:

* Pipeline completo en Mage
* Ejecución correcta
* Resultado final (output)

---

### 4. Documento breve (muy importante)

Formato: Markdown (máx. 2 páginas)

Debe incluir:

#### a) Descripción del pipeline

* Qué hace
* Qué datos usa

#### b) Transformaciones aplicadas

* Explicación clara (no código)
* Justificación

#### c) Resultado obtenido

* Qué información aporta
* Posibles usos

---

## ⚙️ Requisitos técnicos

✔ Uso de `pandas` o similar
✔ Pipeline funcional (sin errores)
✔ Código limpio y estructurado
✔ Nombres de bloques coherentes

---

## 🚫 Errores graves (penalización directa)

* Pipeline que no ejecuta
* Código copiado sin entender
* No incluir evidencias
* No entregar documento

---

* Formato:

  * `.zip` o repositorio
