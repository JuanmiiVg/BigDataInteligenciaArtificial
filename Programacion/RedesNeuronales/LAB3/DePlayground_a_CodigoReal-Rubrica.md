# 🧪 RÚBRICA — LABORATORIO 3

## Laboratorio Puente — Playground → Código Real

### Peso recomendado: 45% de la UD4

(Es el más completo técnicamente.)

---

## 🔹 Criterio 1 — Implementación funcional (30%)

| Nivel | Descripción                                                 |
| ----- | ----------------------------------------------------------- |
| 10    | Modelo correctamente implementado, entrenado y reproducible |
| 8     | Funciona con pequeños errores                               |
| 6     | Funciona parcialmente                                       |
| 4     | Implementación incompleta                                   |
| 0     | No funcional                                                |

---

## 🔹 Criterio 2 — Visualización y análisis (25%)

| Nivel | Descripción                                                             |
| ----- | ----------------------------------------------------------------------- |
| 10    | Visualiza pérdida y frontera de decisión correctamente y las interpreta |
| 8     | Visualización correcta con análisis básico                              |
| 6     | Visualización incompleta                                                |
| 4     | Gráficas poco interpretadas                                             |
| 0     | Sin visualización útil                                                  |

---

## 🔹 Criterio 3 — Experimentación y comparación (25%)

| Nivel | Descripción                                                                  |
| ----- | ---------------------------------------------------------------------------- |
| 10    | Compara profundidad, activaciones y explica diferencias con criterio técnico |
| 8     | Comparación correcta pero limitada                                           |
| 6     | Comparación básica                                                           |
| 4     | Comparación superficial                                                      |
| 0     | Sin comparación real                                                         |

---

## 🔹 Criterio 4 — Conexión conceptual (10%)

Relaciona resultados con:

* backpropagation
* superficie de pérdida
* capacidad de representación

---

## 🔹 Criterio 5 — Calidad técnica (10%)

* Código limpio
* Estructura clara
* Documentación adecuada

---

### 🎯 RA implicados

* RA2 → desarrollo práctico
* RA1 → comprensión profunda
* Parcialmente RA3 → análisis de comportamiento
